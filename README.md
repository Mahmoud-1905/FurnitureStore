## Project Overview

Smart Furniture Store is a modern e-commerce web application designed to provide a seamless and intelligent shopping experience for furniture products. The platform supports three main user roles:

**Customers:** Browse, search, purchase, and review furniture products easily.

**Sellers:** Manage and showcase their products efficiently, handle inventory, and process orders.

**Administrators:** Monitor users, track sales, view reports, and manage overall system performance through a centralized dashboard.
